# Order Suite

Order Suite is a microservices-based architecture built with Java and Spring Boot, designed for learning, portfolio building, and demonstrating backend engineering best practices.

## Microservices Included
- **product-service** – product management API  
- **auth-service** – authentication and authorization  
- **inventory-service** – stock and inventory control  
- **notification-service** – async notifications (email, queue, events)  
- **gateway** – API gateway and routing  

## Tech Stack
- Java 21
- Spring Boot 3+
- Spring Cloud Gateway
- Docker & Docker Compose
- REST APIs
- JPA/Hibernate
- PostgreSQL
